FileSystem (Version 0.1)
========================

Ziel dieses kleinen Projekts ist eine Medienverwaltung mit SQL-Datenbank-Anbindung. Es sollen Dateien auf dem Server gespeichert und verwaltet werden. Zusätzlich soll es möglich sein, den Dateien zusätzliche Daten wie Kommentare zuzuordnen. Darüber hinaus soll eine Rechteverwaltung berücksichtigt werden, die den Zugriff auf die Dateien steuert.

Für die erste Beta-Version sind also folgende Funktionen angedacht:

- allg. Dateiverwaltung inkl. der üblichen Funktionen
- Dateien-Datenbanktabelle für Zusatzinformationen und -funktionen
- Benutzer- und Rechteverwaltung für das FileSystem
- Suchfunktion
- Kommentar/Bewertungsfunktion für die Dateien
